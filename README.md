# faaaah-extension

plays the faaaah meme sound every time you open a rejection email. logs your rejections. has a reset button for when the pain gets too real.

works on gmail and outlook.

---

## install

1. download and unzip this repo
2. go to `chrome://extensions`
3. turn on developer mode
4. click load unpacked, select the folder
5. done

---

## how it works

opens an email, scans the subject and body for rejection signals â€” formal stuff like "we regret to inform" but also slang like "straight up denied" or "weren't goated enough". if it smells like a rejection, it fires.

when it detects one:
- big red rejected stamp hits the center of your screen
- faaaah plays
- stay strong card slides up
- chill guy music plays after

---

## notes

- everything stays local, nothing is sent anywhere
- click the extension icon to see your rejection count and clear it
- if you open the same rejection email multiple times, it counts each time. that's intentional
